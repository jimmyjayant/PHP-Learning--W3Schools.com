PHP Superglobal - $_GET

Super global variables are built-in variables that are always available in all scopes.

PHP $_GET

PHP $_GET is a PHP super global variable which is used to collect form data after submitting a HTML form with method="get".
$_GET can also collect data sent in the URL.
Assume we have a HTML page that contains a hyperlink with parameters:

<html>
<body>

<a href="19.6. PHP Superglobals-$_GET-Example 1.php?subject=PHP&web=W3schools.com">Test $GET</a>

</body>
</html>


When a user clicks on the link "TEST $GET", the parameters "subject" and "web" are sent to "19.6. PHP Superglobals-$_GET-Example 1.php", and you can then access their values in "19.6. PHP Superglobals-$_GET-Example 1.php" with $_GET.
